# Martin Dutton Clinical Psychologist App

## Overview

A Progressive Web App (PWA) designed for Martin Dutton Clinical Psychologist.

The app provides:

* Practice announcements
* Blog articles
* Appointment booking via RecoMed
* Contact options
* Crisis support information
* Push notifications
* Admin announcement management

The application is designed to be installable on Android and iPhone devices without requiring distribution through Google Play or the Apple App Store.
