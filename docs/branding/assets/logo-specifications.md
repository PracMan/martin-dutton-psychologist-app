# 1. Purpose

This document defines the technical specifications for all approved Martin Dutton Clinical Psychologist App logo assets.

The objective is to ensure:

* Consistent branding
* Correct exports
* High-quality rendering
* Scalable implementation
* Cross-platform compatibility

---

# 2. Approved Logo Assets

Version 1 includes three official logo variations:

1. Semicolon Logo
2. Heartbeat Logo
3. Full Horizontal Logo

No additional logo variations may be created without approval.

# 6. Clear Space Rules

All logos require minimum clear space.

Definition:

```text
1 × semicolon height
```

No UI elements may enter the clear space area.

---

# 7. Colour Rules

Approved Brand Blue:

```css
#2E74B5
```

---

Approved Text:

```css
#333333
```

---

Approved Secondary Text:

```css
#666666
```

---

Not Permitted:

* Alternative colours
* Gradients
* Shadows
* Outlines
* Rotations

Exception:

Approved glassmorphism app icon.

---

# 8. Background Rules

Approved Backgrounds:

* Transparent
* White
* Warm White (#FCFCFA)

---

Not Approved:

* Busy photography
* Patterned backgrounds
* Low-contrast surfaces

---

# 9. File Naming Standards

```text
semicolon.svg
semicolon.png

heartbeat-logo.svg
heartbeat-logo.png

full-logo-horizontal.svg
full-logo-horizontal.png
```

All filenames lowercase.

Use hyphens only.

# 11. Accessibility

Requirements:

* High contrast
* SVG preferred
* Sharp rendering on Retina displays
* Scalable without loss of quality

---

# 12. Acceptance Criteria

Logo package is complete when:

* All SVG files created
* All PNG files exported
* Transparent backgrounds verified
* Branding colours verified
* Assets committed to repository
