# 1. Brand Overview

The Martin Dutton Clinical Psychologist App is designed to communicate:

* Hope
* Professionalism
* Calmness
* Trust
* Accessibility

The brand should feel modern and supportive without appearing clinical or corporate.

---

# 2. Primary Brand Identity

## Primary Logo

Blue Semicolon

Purpose:

Primary application identity.

Used for:

* App Icon
* Splash Screen
* Login Screen
* Home Screen Header
* PWA Installation Assets

---

# 3. Secondary Brand Identity

## Heartbeat + Semicolon Logo

Purpose:

Extended practice branding.

Used for:

* Website
* Marketing Material
* Documentation
* Social Media Assets

Not used as the primary mobile application icon.

---

# 4. Brand Personality

The application should feel:

### Calm

Simple layouts and generous spacing.

---

### Professional

Clean typography and restrained colour usage.

---

### Supportive

Clear messaging and approachable interactions.

---

### Modern

Contemporary UI patterns and mobile-first design.

---

# 5. Visual Style

Design Language:

* Minimalist
* Spacious
* Soft shadows
* Rounded corners
* Limited colour palette

Avoid:

* Heavy gradients
* Aggressive colours
* Busy backgrounds
* Complex animations

---

Used throughout:

---

# 9. Tone of Voice

Communication should be:

* Clear
* Respectful
* Professional
* Supportive

Avoid:

* Medical jargon
* Alarmist language
* Corporate terminology

---

# 10. Accessibility

Brand implementation must comply with:

WCAG AA

Requirements:

* Adequate contrast
* Readable typography
* Accessible interactive elements
