# 1. Overview

This document defines the approved usage of all Martin Dutton Clinical Psychologist App logos and branding assets.

The purpose is to ensure:

* Consistency
* Professional appearance
* Correct sizing
* Correct spacing
* Accessibility

Only approved logo files may be used.

# 12. Accessibility

Requirements:

* Sufficient contrast
* High resolution assets
* SVG preferred where possible
* Alternative text provided

---

# 13. Prohibited Usage

Do NOT:

* Stretch logos
* Distort proportions
* Change colours
* Rotate logos
* Add effects not approved
* Overlay logos on busy backgrounds

---

# 14. Acceptance Criteria

Implementation is complete when:

* Correct logo used throughout app
* App icon exported correctly
* Splash assets created
* SVG master files stored
* Branding remains consistent
